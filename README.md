# Study-Tracker
A webapp to help you manage your studies
(Using reactjs as frontend and AppWrite as backend service)
<br><br>
<b>Features:</b>
<ul>
  <li> Authentication Page (under development)
  <li> Note Maker
  <li> TodoList
  <li> Important Dates Manager/Reminder
  <li> StopWatch
  <li> Motivational quotes
</ul>
